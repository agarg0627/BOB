Keyboard shortcuts

BOB has three built-in shortcuts. Click a row, then press the combo you'd like to use. They work on any webpage you're browsing.

Open BOB
Pop up the prompt box on any page so you can build a new feature.

Refine last feature
Open the most recent feature for this site so you can tweak it.

Quick-toggle bar
Show a small panel listing this site's features so you can flip them on or off.

Tip: include at least one of Ctrl, ⌘, Shift, or Alt so the combo doesn't conflict with normal typing. Press Esc while editing to cancel.
