Overlay Sandbox

Press ⌘K / Ctrl+K to toggle the overlay.

Test triggers (mock callbacks):

  • Type a prompt containing error-gen to trigger an onGenerate rejection (error appears in the prompt area).
  • In the preview, edit the name to include error-install and click Install to trigger an onInstall rejection (error appears in the preview area, button re-enables).
  • Click Install (with no error trigger) to see the success toast at bottom-right.
  • Click Cancel to return to the prompt with the original text pre-filled.
  • Press Esc from any state to close.
  • In the preview, press ⌘↵ / Ctrl+↵ to install.
